# 档案服务验收单应用插件

用于在飞书多维表格应用插件中选择一个数据源和记录，生成可打印的档案服务验收单。项目名称、服务起止时间和服务内容来自选中的记录；数量和单价仅在插件面板中编辑，金额与费用合计自动重算。

## 记录字段约定

默认自动匹配以下字段名称，配置面板可改为其他字段：

- `项目名称`
- `服务起始时间`、`服务截止时间`，或单个 `服务起止时间`
- `服务内容`、`单位`、`数量`、`销售单价`

服务内容、单位、数量、销售单价均可在同一记录中保存多个值，以英文或中文逗号分隔。例如：

```text
服务内容：文书档案整理,业务档案整理,档案盒
单位：件,卷,只
数量：1471,35,61
销售单价：0.52,12,8
```

插件按各字段的相同位置组合为明细，并按 `1..N` 自动编号。服务内容不足的条目会标记为“未命名服务”，数量和单价仍可在插件中补录或调整。

## 本地运行

```bash
pnpm install
pnpm dev
```

未运行在飞书应用插件容器时，应用使用内置演示记录，便于核对打印版式。

## 构建与 Gitee Pages

```bash
pnpm build
```

构建产物为 `dist/`。推送到 Gitee 后，在仓库的 Pages/静态网站托管设置中选择发布分支与 `dist` 目录；生成的 HTTPS 地址填入飞书应用插件的运行地址。`vite.config.js` 已设置 `base: './'`，适合静态目录部署。

应用插件上线前还应在飞书中验证 Base 选择、Create/Config/View 状态、字段映射和打印结果。
